Datenschutz

Allgemeines

Diese Website dient als Bewerbungs- und Portfolioseite. Es gibt kein eigenes Kontaktformular und keine Übertragung von Formulardaten an einen eigenen Server.

Das Hosting erfolgt über GitHub Pages. Beim Aufruf der Website können durch den Hosting-Anbieter technische Zugriffsdaten verarbeitet werden.

Lokale Datenverarbeitung

  • Diese Website speichert keine Bewerbungsdaten in einer eigenen Datenbank.
  • Es werden auf der aktuellen Version keine Formulardaten lokal im Browser gesammelt.
  • Statische Dateien wie PDF, HTML, CSS, SVG und XLSX werden direkt über GitHub Pages ausgeliefert.

Externe Dienste

Auf der aktuellen Bewerbungswebsite werden keine externen Zahlungsdienste oder Newsletter-Dienste eingebunden.

Illustrationen und Grafiken sind lokal in die Website eingebunden.
