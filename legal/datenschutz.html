Datenschutz

Allgemeines

Diese Website dient als Bewerbungs- und Portfolioseite. Es werden keine Kontaktformulare an einen eigenen Server uebermittelt.

Hosting erfolgt ueber GitHub Pages. Beim Aufruf der Website koennen durch den Hosting-Anbieter technische Zugriffsdaten verarbeitet werden.

Innerhalb der Shop-Demo wird der Warenkorb ausschliesslich lokal im Browser ueber `localStorage` gespeichert.

Welche Daten in der Demo lokal genutzt werden

  • Warenkorbpositionen in der Shop-Demo
  • Produktmengen fuer den Demo-Checkout
  • Keine Uebertragung dieser Daten an einen eigenen Server

Externe Dienste

Ein echter Stripe-Checkout wird nur dann aufgerufen, wenn in der Shop-Demo ein Stripe Payment Link hinterlegt wurde. In diesem Fall erfolgt die Weiterleitung direkt zu Stripe.

Bildmaterial stammt von Pexels und ist lokal in die Website eingebunden.
