Datenschutz

Allgemeines

Diese Website dient als Bewerbungs- und Portfolioseite. Es gibt kein eigenes Kontaktformular und keine Übertragung von Formulardaten an einen eigenen Server.

Das Hosting erfolgt über GitHub Pages. Beim Aufruf der Website können durch den Hosting-Anbieter technische Zugriffsdaten verarbeitet werden.

Lokale Daten in der Shop-Demo

  • Warenkorbpositionen werden lokal im Browser gespeichert.
  • Die Speicherung erfolgt ausschließlich über localStorage.
  • Es findet keine Übertragung dieser Demo-Daten an einen eigenen Server statt.

Externe Dienste

Ein echter Stripe-Checkout wird nur dann aufgerufen, wenn auf der Checkout-Seite ein Stripe Payment Link hinterlegt wurde. In diesem Fall erfolgt die Weiterleitung direkt zu Stripe.

Bildmaterial stammt von Pexels und ist lokal in die Website eingebunden.
